

# Farbe

An interface for rendering over OpenGL ES 2 and GLFW. It comes with an eDSL for writing shaders and simple combinators for rendering.

[haddock docs](https://vekkq.github.io/farbe/dist-newstyle/build/x86_64-windows/ghc-9.12.1/farbe-0.1.0.0/doc/html/farbe/Graphics-Farbe.html).
